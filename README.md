# JavaScript Fundamentos

Projeto desenvolvido durante meus estudos para praticar conceitos fundamentais de JavaScript utilizando HTML e CSS.

## Objetivo

Aplicar conceitos básicos de JavaScript por meio da interação com elementos da página, utilizando funções, eventos e manipulação do DOM.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Exibição de uma mensagem de boas-vindas.
- Personalização da saudação ao clicar em um botão.

## Como executar

1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador.

## Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

- Estruturação de páginas com HTML5.
- Estilização utilizando CSS3.
- Criação de funções em JavaScript.
- Manipulação do DOM.
- Eventos de clique.
